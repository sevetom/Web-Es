Alla base del web sono definiti tre standard fondamentali:
- HTML (Hypertext Markup Language): è un linguaggio di markup utilizzato per l'identificazione del formato delle risorse delle pagine web, fornendo i mezzi per definire la sua 	struttura e layout. Le pagine web in cui viene utilizzato HTML fanno uso di elementi rappresentati da tag che definiscono il significato e l'aspetto dei contenuti. Il 	browser interpreta poi questa linguaggio e mostra la pagina all'utente nel suo aspetto desiderato.

- HTTP (Hypertext Transfer Protocol): è il protocollo di comunicazione che permette lo scambio e il trasferimento di informazioni sul web. Questo protocollo è alla base del 	trasferimento di dati tra il client e il server consentendo la visualizzazione delle pagine web nel momento in cui l'utente desidera visitarla.

- URI (Uniform Resource Identifier): è una stringa di caratteri che identifica un nome o una risorsa su internet in modo univoco. Gli URI includono due sottocategorie: 
	- URL (Uniform Resource Locator): specifica l'indirizzo di una risorsa su internet, come gli indirizzi delle pagine web.
	- URN (Uniform Resource Name): identifica una risorsa in modo persistente senza dover specificare la sua posizione.
